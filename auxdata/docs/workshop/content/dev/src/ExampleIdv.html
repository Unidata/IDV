We will be using an example application in <%workshop.installdir%>/ucar/unidata/apps/example
  • Go to the example directory: [ht::cmdline {cd <%workshop.installdir%>/ucar/unidata/apps/example}]

  • To run the ExampleIdv: [ht::cmdline {runExample.sh}] or do: [ht::cmdline {java -Xmx512m ucar.unidata.apps.example.ExampleIdv}]
