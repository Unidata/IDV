Installing and Running the Unidata IDV with Web Start

This web page explains how to install and run the Unidata Integrated Data Viewer (IDV) using Java Web Start. If you already have Web Start installed on your machine then click here to launch the IDV.

The Unidata IDV is the reference application for the Integrated Data Viewer Framework making general tools for displaying geo-referenced datasets. See the User Guide for an overview of the IDV. Java Web Start is an application-deployment technology that downloads all software needed from the Web and then starts the application. You do not need to set up directories, or FTP files, or configure anything - Web Start does it for you.

To run the IDV through Java Web Start just follow these steps:

  1. Read the system requirements.
    Before starting, please see the System Requirements to ensure that your machine is capable of running the IDV.

  2. Install Web Start
    • If you are Windows, Solaris/SPARC or Linux, go to the Sun J2SE download page. Download and install the latest version of the Java Runtime Environment (JRE) for your platform. The JRE includes Web Start. All you need is the JRE. (You don't need the SDK, NetBeans or any of the other myriad of options Sun provides; they are mainly for Java developers.) Depending on your platform, after clicking on the latest JRE download option, you might get a license agreement screen. If you accept the terms, it will take you to the actual download page where you click on the download appropriate for your platform. The download page explains different options for "online" or "offline" installation, etc. But, in general, the installation process will be similar to that for installing other software packages on your machine. If you are running Windows, Web Start will download everything you need.
    • If you are running Mac OS-X (version 10.3.1 or later), Web Start is already installed. If you are running on a version less than 10.4.0 you will need to download and install the Java 3D Update available here.
    • For other systems, Web Start is not available, so go to the Downloading and Running the IDVsection of the Users Guide and use either the InstallAnywhere installers or the manual installation.

  3. Run the IDV
    Click here to launch the IDV. Once Web Start is installed, you only click here to start the IDV every time.
For additional information on starting the IDV through Web Start, see the IDV User's Guide and the FAQ.
