[ht::head {WikiFolder}] Wiki Text in Other Entries You can use the wiki text display capabilities to customize the default HTML view of any other entry or folder types. If the description of the entry begins with "<wiki>" then we treat the description as wiki text and process it to create the default HTML view. This allows you to override the default html display and come up with your own display.
  • Create a wiki text for the workshop data folder
    • Navigate to RAMADDA data folder, create a new folder named idvData
    • In the idvData folder, create a folder named Pictures, and upload a few idv generated images.
    • In the idvData folder, create a folder named Schedule, inside Schedule create four subfolders with each of this workshop date assigned to the date range of the subfolder: Monday, Tuesday, Wednesday, and Thursday.
    • Back to idvData folder, Go to the [ht::menu Edit {Edit Entry}] menu, and copy the follow text into the description box: 
                  <wiki>
            <center>
            <h3>
            Welcome to the 2012 IDV/RAMADDA User's Training Workshop.
            </h3>
            </center>

            <table width="100%"><tr valign="top"><td>

            {{tree title="Links"}}
            </td><td>
            <img src="http://www.unidata.ucar.edu/photos/2006userwkshp/collage.jpg" width="500">
            </td></tr></table>

            Presentation Schedule:
            {{import entry=entryId output="calendar.calendar" }}
      The entryId can be obtained by clicking on the "Add entry id" button on the upper right conner and go through the tree structure to get to the Schedule folder.
    • Click on the Save button.
