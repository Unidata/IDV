Hovmöller diagrams excel at displaying data as a function of time and location.

The Hovmöller diagram was invented by Danish meteorologist Ernest Aabo Hovmöller (1912-2008). Typically, time is displayed on the ordinate (y-axis) and geolocation on the abscissa (x-axis). They are commonly used in meteorology and oceanography.

Let's create a Hovmöller of temperature

  • In the [ht::dialog Field Selector]'s [ht::command Data Sources] panel, select the [ht::dataset NCEP Model Data ETA data source] that you loaded in during the previous exercise.
  • In the [ht::command Fields] panel, expand the [ht::param 3D grid] tab. Select the [ht::param temperature] field.
  • In the [ht::command Displays] panel, expand the [ht::param Hovmoller] tab and select [ht::display Time-Latitude (Color Shaded)] then click the [ht::command Create Display] button.
  • Modify the color table by clicking the Temperature button. Change the range from -30 to 30.
  • Can you distinguish a pattern in the data?
