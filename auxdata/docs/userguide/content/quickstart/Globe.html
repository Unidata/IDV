The illustration below shows an IDV Globe Display of GFS numerical weather prediction model output of mean sea level pressure (as color-shaded image and contour lines) and 50 m/s wind speed isosurfaces showing the jet streams. [ht::cimg ../examples/images/GlobeDisplay.gif {Globe Display}] In the Globe Display version of the IDV, the displays and maps are projected onto a spherical globe. The globe can be rotated by hand or automatically, along with the usual zooming and time animation of displays on the globe.

To create a Globe View window, use the [ug::menu File New {View Window} {Globe Display} {One Pane}] menu. Uncheck the [ug::menu View Show {Wireframe Box}] menu to turn off the box in the display. This section describes how to make plots of global satellite imagery on the Globe display. The set of steps include: <%childlist%> Choosing Global Satellite Imagery After starting the Globe version of the IDV, select the Image Chooser in the Dashboard.

  • Connect to the adde.ucar.edu server using [ht::command GLOBAL] dataset.

  • From the [ht::command {Image Data Type}] selector, choose the Global Thermal IR composite images type.

  • Click the [ht::command {Add Source}] button to load the data source.
Displaying the data Once the data source has been selected, from the Field Selector you can create the display by doing the following:
  • Select the "Global Thermal IR composite images" data source in the [ht::command {Data Sources}] panel.

  • Select the "Band: 1" field in the [ht::command {Fields}] panel.

  • Select "Image Display" in the [ht::command {Displays}] panel.

  • Click the [ht::command {Create Display}] button to create the display.
Using the display When the data are displayed, the default view will be from the center of the image (0N, 0E). You can move the display using the standard IDV zoom, pan and rotate functions. For this exercise try the following:
  • Use the [ug::menu Projections Viewpoints] menu to change the viewpoint to an are of interest.

  • Start the image animation using the Animation Widget.

  • Use the [ug::menu View Viewpoint] menu or the Viewpoint Toolbar to set the view to the Western Hemisphere.

  • Use the [ug::menu View Viewpoint {Auto-rotate View}] menu. Check this on to spin the globe along the vertical axis. You can right click on the Viewpoint Toolbar's Auto-rotate button to change the speed of rotation.
  • Click anywhere in the view to stop the rotation.
Any suitable data with navigation information (latitude, longitude, and altitude) that the IDV can handle can be plotted on the globe display. Review the section on Plots from Gridded Data.
