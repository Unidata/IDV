The IDV can create gridded data fields from point data using Barnes Objective Analysis.
  • Leave the metar.csv point data source from the prior exercise.
  • In the Field Selector open the "Gridded Fields" tab. This lists is the numeric fields in the data.
  • Choose "T" and create a [ht::display Color-Shaded Plan View] display.
  • In the [ht::command Grid Parameters] tab check off [ht::button Use Default] and change the number of passes to 6 and create another [ht::display Color-Shaded Plan View] display.
  • Toggle the visibility to compare the differences.
  • Create a [ht::display Point Data Plot] display and change its Layout Model to "Temperature" and compare the values to the grid values.
