Publishing Content to RAMADDA via a Web Browser.

So far, we have mainly been publishing data through the IDV. Users can also publish data directly via the RAMADDA website.

  1. Go to your RAMADDA Instance and navigate to your Users folder.
  2. Create a new entry by clicking the plus sign. You will have a panoply of choices. Select file.
  3. Enter information about the file you wish to save. For the purposes of this exercise point enter the following URL in the URL tab: http://motherlode.ucar.edu/thredds/fileServer/galeon/testdata/RUC.nc
  4. You can examine the netCDF metdata by going to [ht::menu Data {File Metadata}])
  5. You can visualize the netCDF metdata by going to [ht::menu Data {Grid Displays}])
  6. Select the field you would like to display.
  7. Add additional information to your image such as "World Country Outlines".
