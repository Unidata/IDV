Below the main menus you will find the Main Toolbar. [ht::cimg images/MainToolBar.gif {Main Toolbar}] Many of the more commonly used features of the IDV can be accessed here. Hold your mouse over a tool bar button to get a popup tip on what that button does. The tool tip will also appear in the Tooltip panel near the bottom of the main window.

The Main Toolbar is user configurable. See the section on setting User Preferences for more information.
