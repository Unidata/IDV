Getting your WRF post-processed data into the IDV environment.

The IDV GUI has two main windows: the [ht::popup "http://www.unidata.ucar.edu/software/idv/docs/userguide/quickstart/index.html" "Dashboard and the View Window"]. The View Window is where you will see and manipulate displays that you create. The Dashboard is where you will be able to load your data and easily access all controls for manipulating the properties of variables you wish to display (like setting contour properties, adding color scales, changing color tables, etc). In order to get your WRF output into the IDV from the GUI interface, you will need to start the IDV and begin work with the Dashboard:

[ht::cimg images/Dashboard.png]

The display above shows the Dashboard window of the IDV. The Data Choosers tab has been selected, and we are going to load Files off of our disk (note "Files" under "General" is selected on the left hand side of the Dashboard). Follow these steps to begin the process of loading your data into the IDV.

  1. Locate your WRF output in the file browser and select the files you wish to load (hold shift to select multiple files)
  2. Make sure that "Data Source Type" is set to "Aggregate Grids by Time"
  3. Click "Add Source"

Once you click "Add Source", the IDV will switch you from the Data Choosers tab to the [ht::popup "http://www.unidata.ucar.edu/software/idv/docs/userguide/data/FieldSelector.html" "Field Selector"] tab:

[ht::cimg images/FieldSelector.png]

The Field Selector tab is where you will select the variables and types of images to generate. In the screenshot above, the 2D variable [ht::param Pressure_reduced_to_MSL @ msl] (left side of the Field Selector tab) has been selected, and a display type [ht::menu {Plan Views} {Contour Plan View}] (top right side of the [ht::command Field Selector] tab) has been chosen. Note that this is the same view that was generated in the previous Jython examples. For now, we will leave spatial and temporal subsetting as default (i.e. no subsetting) and load the entire grid (bottom right panel in the Field Selector tab). For more information on subsetting, see the [ht::popup "http://www.unidata.ucar.edu/software/idv/docs/userguide/data/FieldSelector.html" "Field Selector"] page. The final step is to push the [ht::command Create Display] button. Note - up until this point, very little information has been loaded off of your disk into memory. Once you press the [ht::command Create Display] button, data for the variable you selected will be read off of disk - depending on the size of your data set, this may take quite some time!

Once the data has been loaded, the IDV will generate an image in the View Window. Congratulations, you've just loaded your WRF data into the IDV!
