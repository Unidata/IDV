The Parameter Groups Editor is opened through the Tools menu with [ug::menu Tools {Parameter Groups Editor}] and allows for the specification of "parameter groups", named groups of parameter names. [ht::cimg images/ParameterGroupsEditor.gif {Parameter Groups Editor}]

There are two sets of defaults: user defaults in the tab User Groups, and system groups supplied with the IDV in the System Groups tab. Look at the system defaults for examples.

The system settings cannot be edited by the user, but right clicking a row allows you to copy a group to the "user defined" table. When you do this the Parameter Groups Properties window allows you to define the sets of parameters that fall under this group.
