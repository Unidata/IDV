This section describes how to make a display using Level III radar data. [ht::cimg images/Level3Display.gif] The steps include: <%childlist%> Choosing Radar Image Data from a Remote Server In the Dashboard window select the Data Chooser tab. From here select the Radar tab to view the Level III Radar Chooser: [ht::cimg ../data/choosers/images/Level3Chooser.gif]
    [ug::step "Choose an ADDE [ug::menu Server] and a [ug::menu Dataset] to use:" ../data/choosers/images/RadarStep1.gif { Try Server: adde.ucar.edu and Dataset: RTNEXRAD. }] [ug::step "Press [ug::button Connect] to query the server for available radars types:" ../data/choosers/images/RadarStep2.gif] [ug::step "The map shows radar station locations and IDs. Select one by clicking on it." ../data/choosers/images/RadarStep3.gif { The Declutter check box allows you to show all stations (not checked), or only a limited number of stations that do not overlap each other (checked). You will need to zoom in to see all the stations without overlaps. }] [ug::step "For the Radar Data Type of data from this radar, select a radar product, such as Base Reflectivity Tilt 1, from the pull-down menu." ../data/choosers/images/RadarStep4.gif] [ug::step {Either, choose a relative set of times steps (e.g., Last 5 times):} ../data/choosers/images/ImageStep4.gif] [ug::step {Or, choose a set of absolute times:} ../data/choosers/images/ImageStep5.gif { When choosing absolute times for the first time the IDV needs to query the ADDE server for the times. This may take some time. To select more than one time use Control-Click or Shift-Click. }] [ug::step "When done, load the selected radar data with the [ug::button Add Source] button." ../data/choosers/images/ImageStep6.gif \ "If the [ug::button Create display] button is selected then a radar display will automatically be created"]
Selecting the Radar display to be made The radar data source that you selected will be shown in the Field Selector window. [ht::cimg images/RadarSelector.gif] If you checked off the [ug::button Create display] button or if you want to create another radar display click "Image Sequence" in the Field panel and click on "Image display" in the Displays panel. Make the display by clicking on Create Display.

You can add Radar Range Rings to the display. To control time animation use the Time Animation Widget.
