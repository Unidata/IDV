You can create a default IDV state and save it off using the bundle facility.
  1. By default the IDV imports a [ht::filename Default] bundle when it starts up.
  2. Open up a new view window with the [ht::menu File New {View Window}] menu item.
  3. Change its state.
    • Change its perspective.
    • Select a different projection with the [ht::menu Projections Set] menu.
  4. Save the current state as the default bundle by selecting the [ht::menu File {Default Bundle} {Save}] menu item from the main menu bar.
  5. This file is saved in the user's local resource directory:
    <home directory>/.metapps/DefaultIdv
    (e.g., <%workshop.homedir%>/.metapps/DefaultIdv) as [ht::filename default.xidv].
  6. Exit the IDV and restart it. You should see the previously saved state.
  7. Remove all data and displays.
  8. Select the [ht::menu File {Default Bundle} Open] menu item or the button from the toolbar to reload the bundle.
  9. Delete the default bundle with the [ht::menu File {Default Bundle} {Remove}] menu item from the main menu bar.
  10. Exit the IDV and restart it. You should see the previously saved state defined in your user preferences.
