The IDV can directly write out Webstart JNLP files
  1. First, load in a data source and create a display.
    • [ht::loadBasicGrid]
    • [ht::createBasicDisplay -display {Color-Shaded Plan View}]
  2. Save the bundle with [ht::menu File {Save As}].
  3. Make sure you have all right side buttons turned on from the prior exercise.
  4. Select [ht::filename workshop.jnlp] as the file and press [ht::button Save].
  5. From the JNLP dialog press [ht::button OK].
  6. Now, lets look at this [ht::filename workshop.jnlp] file. From a terminal window type:
    less workshop.jnlp
  7. Back to the IDV save the bundle now as [ht::filename workshop2.jnlp].
  8. From the JNLP dialog turn off [ht::button Include bundle in JNLP file] and enter some base url in the text field. e.g., http://www.unidata.ucar.edu. Press [ht::button OK].
  9. From a terminal window type:
    less workshop2.jnlp
