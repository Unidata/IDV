Learn to do more with NEXRAD Data Displays

  1. Access realtime NEXRAD data from the file system from the Radar Data Source Chooser
    • [ht::openDataSourceChooser]
    • In the [ht::dialog Data Source Chooser] click on the [ht::tab Radar] tab. Then click on the [ht::tab NEXRAD Local] tab to see the NEXARD radar data chooser panel.
    • From the [ht::command Relative Times] selector, select the [ht::command Use Most Recent 5 files].
    • Use the directory (folder) selector to find the radar station data you want.
      • Go to the directory where the data are stored (e.g., [ht::filename /data/gempak/nexrad/craft]).
      • Pick a station that has weather today.
      • Click the [ht::button {Add Source}] button.
    • [ht::editDataSourceProperties]
    • In the [ht::command Polling] section, check the [ht::command Active] option and set the [ht::command Check every] box to 5 minutes, then click [ht::button OK].
    • Select the [ht::param Reflectivity] field and the [ht::display Radar Sweep View in 2D] display, then click [ht::button Create Display].
    • Wait for 5 minutes and see that the display reloads with new data.
  2. Access realtime and casestudies Level3 data from the remote server from the Radar Data Source Chooser
    • [ht::openDataSourceChooser]
    • In the [ht::dialog Data Source Chooser] click on the [ht::tab Radar] tab. Then click on the [ht::tab NEXRAD Remote] tab to see the NEXRAD radar data remote chooser panel.
    • In the catalog selector drop-down, select (or type in) the motherlode catalog this is included. In the collections drop-down, select [ht::tab Level III Radar for Case Study CCS039]
      • Select a product from [ht::tab product] drop down selector, or the first product will be selected
      • Zoom in the map and select the station KDVN.
      • Click on the Absolute time radio button. Select the images from 1998-06-29 21:00:00Z through 1998-06-29 21:59:00Z.
      • Click the [ht::button {Add Source}] button.
    • Display the casestudies Level 3 datasets.
    • In the collections drop-down, select [ht::tab Level III Radar from IDD]
    • Display the real time Level 3 datasets.
  3. Multiple Radars
    • Load Level II data for three or four times from three adjacent stations which have some detectable signals over the combined area.
    • Make 2D sweeps plots from all stations. Experiment to see if the order you display data from stations makes a difference.
    • Display the same data with 3D sweeps and rotate the view window to see intersecting sweeps.
  4. Multiple Data Sources
    • Select a satellite data source that covers a region of the US that includes radars with echos.
    • Overlay the satellite image with Level II radar data.
