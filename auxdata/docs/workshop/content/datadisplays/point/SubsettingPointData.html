You can do time binning and spatial subsetting for point data.
  1. [ht::clearDisplaysAndData]
  2. Load in the file [ht::filename <%workshop.datadir%>/point/madis.nc]. Make sure you the Data Source type select "netCDF Point Data files"
  3. Create a Point Data Plot. Say Yes to seeing all of the times. Notice, the number of times. Not very workable.
  4. In the display control window go to the [ht::command Times] tab. Turn on the [ht::command enabled] button and enter 5 minutes for the [ht::command Only Show Every]. Press return.
  5. Note: this isn't time binning. This just skips the obs.
  6. Remove that display.
  7. Now we're going to bin the times.
    • Bring up the Properties dialog for this madis point data. (Double click or right click on the data source entry in the Field Selector).
    • Choose a 15 minute [ht::command Bin Size] and press [ht::button OK].
    • Now, create a point data plot again.
  8. Spatially subset the point data.
    • Bring up the Properties dialog for the madis point data. Go to the [ht::command Spatial Subset] tab.
    • Zoom in over Colorado and drag a box with the left mouse to select a region.
    • Turn on [ht::button Reload Displays] and press [ht::button OK].
