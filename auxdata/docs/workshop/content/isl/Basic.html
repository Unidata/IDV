Create a simple "Hello World" ISL file.
  1. First of all bring up a terminal window and create a sub-directory: [ht::filename isl]
  2. Change directory to the [ht::filename isl] directory (cd isl).
  3. Copy the following to your [ht::filename isl] directory. Right click on link and do 'Save Link As' [gen::isl basic1.isl]
  4. From the terminal run the IDV with the basic1.isl as the command line argument: 
    idv basic1.isl
  5. Lets loop this now by adding a [gen::islTag group] tag. Copy this file: [gen::isl basic2.isl]
  6. Let's define some properties with the [gen::islTag property] tag. Properties are referenced as \${propertyname}. [gen::isl basic3.isl]
