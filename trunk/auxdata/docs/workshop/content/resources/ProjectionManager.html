In this section we will cover how to create and edit map projections.
  1. Select the [ht::menu Projections New/Edit] menu item.
  2. View an existing projection.
    • Select "US>CONUS" and press [ht::button Edit]
    • Right mouse/drag in map to scroll. Shift+Left mouse drag to zoom.
    • Press [ht::button Cancel]
  3. Create a new projection.
    • Click on the [ht::button New] button.
    • In the [ht::dialog Define/Edit Projection] window enter the name [ht::qt {Workshop>Colorado}] in the [ht::command Name] box.
    • Leave the [ht::command Type] as [ht::qt LatLon].
    • Click in the map view on the left and delete the existing bounding box by pressing the [ht::command Delete] key.
    • Click in the [ht::menu Maps] menu and turn on the [ht::command World Coastlines] map.
    • Zoom in over the United States by holding the Shift key down and dragging the mouse with the left mouse button down.
    • Draw a bounding box over Colorado by dragging with the left mouse button pressed. You can adjust the size of the box by dragging on the boxes on the box. You can move the box by clicking and dragging on the box lines. When you are finished, click the [ht::button Save] button.
    • Back in the [ht::menu Projections Set] menu select [ht::menu Workshop Colorado] to view the projection.
  4. Set the default projection to "US>CONUS"
    • Open the Preferences Manager from the [ht::menu Edit Preferences] menu.
    • Select the [ht::command {Main View}] tab.
    • Select the [ht::command {US>CONUS}] projection in the [ht::command Default Projection] dropdown list.
    • Click the OK button.
    • The next time you start the IDV, that will be your default projection.
