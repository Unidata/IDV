Version <%idv.version%>
The items below reflect changes since the 2.3 release. Users can save properties of display controls in favorite display settings. These can be used when displays are created or shared between controls. See the section on Display Settings in the User's Guide. The Jython library has been enhanced with many new functions. Most of the GEMPAK grid diagnostics have been implemented. For a complete list, see Jython Library Functions. A new interactive Jython Shell can be accessed under the [ht::menu Edit Formulas {Jython Shell}] menu. This shell allows you to interactively write new procedures, load in data and create displays. Various and sundry bugs were fixed in this release.

For a list of outstanding known problems, see the Known Problems page.
