Clicking on the main menu bar choice [ug::menu Help {Support Request Form...}] shows a form to post a support request to the Unidata Program Center (UPC). This form can also be brought up from the Error Dialog. [ht::cimg images/SupportRequestForm.gif {Support Request Form}] If you encounter a problem or have a comment to share, you can use this form to do so.

By checking on the [ug::button Include Current State as Bundle] button the current state of the IDV will be submitted along with your request. You can also add attachments to your submission.
