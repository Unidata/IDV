NASA'S WorldWind program comes with a tiled location database that allows for progressive disclosure of locations as you zoom in. The IDV makes use of this database.
  1. Start fresh by clearing all displays and data [ht::refresh] and go to a CONUS projection ([ht::menu Projections Set US CONUS] menu item).
  2. Create the [ht::display WorldWind] display control from the [ht::menu Displays Special {WorldWind Locations}] menu item.
  3. This is just like the [ht::display Location ] display control except it progressively discloses more detailed locations as you zoom.
  4. Zoom into Colorado and watch how the display changes.
  5. Zoom into Boulder. Remember, if things go blank you just zoomed past the surface of the Earth. Use the [ht::menu Projections Set {Use Displayed Area}] menu item to reset the projection.
  6. Add a [ht::display Background Image] and change the layer to [ht::param Urban Area].
  7. Keep zooming.
  8. Remove the [ht::display Background Image] display control.
  9. Change projections to some other part of the world and see what is listed.
