Saving the displays created in the IDV.

In the previous section, we loaded some WRF output and created a very simple contour display. In this section, we will explore how to save images and animations of our displays. However, before proceeding, let's create a [ht::param multipanel display] which extends our previous display of [ht::param Pressure_reduced_to_MSL @ msl].

  1. From the [ht::command Dashboard], click [ht::menu {File} {New} {View Window} {Map View} {Two Panes}].
  2. Go to the [ht::command View Window] and click on the left display (look for border around the window to turn blue).
  3. Go back to the [ht::command Field Selector] in the Dashboard and create a [ht::display Color-filled Contour Plan View] of [ht::param Pressure_reduced_to_MSL @ msl].
  4. Go to the [ht::command View Window] and click on the right display (again, look for border around the window to turn blue).
  5. Go back to the [ht::command Field Selector] in the Dashboard and create a [ht::display Color-filled Contour Plan View] of [ht::param Temperature @ Surface].

Now, let's save both images:

  1. From the [ht::command View Window], click [ht::menu {View} {Capture} {Image}]. Note that since we are going to capture both images, it does not matter from which display we click [ht::command View]. If you wish to capture a single display, make sure to use the [ht::command View] menu from that display.
  2. An image save dialog box will appear. Give the image a name (TwoDisplays.png), select [ht::param Capture What : All Views], and set the quality to [ht::param High].
[ht::cimg images/InteractiveWRFImageDialog.png]
  1. Click [ht::command Save]. Note that the size of the image will match the size (pixel dimensions) of the window on your display.
[ht::cimg images/InteractiveWRFTwoDisplays.png]

A similar procedure is used to capture an animation of your displays:

  1. From the [ht::command View Window], click [ht::menu {View} {Capture} {Movie}]. This will cause the [ht::command Movie Capture] dialog box to appear.
[ht::cimg images/InteractiveWRFMovieDialog1.png]
  1. Make sure [ht::param What to capture] is set to [ht::param All Views], and click the [ht::command Time Animation] button. The view window will come to the front of you desktop display and animate. Then, a Save dialog box will appear.
[ht::cimg images/InteractiveWRFMovieDialog2.png]
  1. Set the [ht::param Frames per second] to 1, and [ht::param End Frame Pause] to 1.
  2. Select the file format (.gif, .mov, .avi, etc.) and name your file.
  3. Click [ht::command Save].
[ht::cimg images/InteractiveWRFTwoDisplays.gif]

To add axis labels to your images. To display the axis labels:

  1. Select the display to which you wish to add labels and click [ht::menu {View} {Properties}].
  2. Click on the [ht::command Horizontal Scale] tab and turn on the [ht::param Latitude] and [ht::param Longitude] scales by clicking the [ht::command Visible] box under each section, and then click [ht::command Apply].
[ht::cimg images/InteractiveWRFViewProperties.png]
  1. Finally, save your display (either image or movie), and breathe a sigh of relief.
[ht::cimg images/InteractiveWRFTwoDisplaysLabs.gif]
