You can use time matching feature in the IDV to create a time synchronized display with both radar and satellite imagery.

First, load up a couple of satellite images.

  1. [ht::clearDisplaysAndData]
  2. [ht::openDataSourceChooser] Click on the [ht::command Images] node in the chooser tree.
  3. Choose [ht::param adde.ucar.edu] server and [ht::param CCS039] dataset, click the [ht::button Connect] button.
  4. From the [ht::command Image Type:] selector, choose the [ht::param 1 km GOES-8 Visible] type.
  5. Select [ht::listSelect] the 2 images at [ht::param 1998-06-29 21:15:00Z] and [ht::param 1998-06-29 22:15:00Z].
  6. Click the [ht::button Add Source] button.
  7. In the [ht::command Times] panel, select the [ht::command Set As Time Driver] menu item.
  8. Click the [ht::button Create Display] button to load in those two images.
Next, let's select some radar imagery using the driver times from the satellite imagery.
  1. First, let's keep the satellite projection for now. Uncheck the [ht::menu Projections {Auto-set Projection}] menu item.
  2. Open the Level III Data Source Chooser
    • [ht::openDataSourceChooser]
    • In the [ht::dialog Data Source Chooser] click on the [ht::tab Radar] tab. Then click on the [ht::tab NEXRAD Remote] tab to see the Level III radar data remote chooser panel.
    • In the catalog selector drop-down, select the motherlode catalog that is included.

        https://thredds.ucar.edu/thredds/radarServer/catalog.xml

      If it is not in the list, type it into the catalog selector area and press enter. In the collections drop-down, select [ht::tab Level III Radar for Case Study CCS039]
      • Zoom in the map and select the station DVN.
      • Click on the [ht::button {Match Time Driver}] button on the bottom left.
      • Click the [ht::button {Add Source}] button.
    • Select the [ht::param Reflectivity] field and create a [ht::display {Radar Sweep View in 2D}] display.
  3. Let's check the time matching result by bring up the animation timeline: click on [ht::menu View {Animation Timeline}], and compare the time selection from two difference data source.
  4. Remove all displays and data.
[ht::cimg images/TimeMatchingTimeLine.png] Try the following time matching exercises on your own using the real time satellite and radar data.
  1. First, load in the latest 5 GOES East/West IR Composite and set the Satellite imagery as time driver.
  2. Second, find the location of interesting weather.
  3. Then, load in the a few level III radar stations which are close to the area identified in the second step, and check on the [ht::button {Match Time Driver}] button.
  4. Check the [ht::menu View {Animation Timeline}] for the matching result.
  5. Remove all displays and data.
