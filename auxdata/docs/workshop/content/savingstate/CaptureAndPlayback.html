[ht::desc The IDV supports the capture and replay of a variety of user interface events.]
The IDV supports the capture and replay of certain user interface events, including loading data sources, creating and manipulating display controls, and changing viewpoints, maps and projections.
  • First, clear any displays and data. [ht::refresh]
  • Open the [ht::dialog Capture] window by selecting the [ht::menu File {Event Capture}] menu item from the main menu bar.
  • We are going to capture the loading of a data source and the creation of a display.
    • Click the [ht::button Start event capture] button to start the capture.
    • [ht::loadBasicGrid]
    • [ht::createBasicDisplay -display {Color-Shaded Plan View}]
    • If it isn't open, open the control window for the newly created display control.
    • Select a new level in the [ht::command Level] list.
    • In the main map view move around the 3D space.
    • Change the projection in the main map view by selecting the [ht::menu Projections Set World] menu item.
  • Now save this set of events off into a capture file.
    • Click the [ht::button Stop Capture] button.
    • Click on the [ht::button Write Captured Events] button to bring up the [ht::dialog Save] dialog.
    • [ht::save workshop.cpt]
  • Now we are going to replay this capture.
    • First, we need to start fresh. [ht::refresh]
    • Load in the file [ht::filename workshop.cpt] by clicking the [ht::button Load Captured Events] button in the [ht::dialog Capture] window.
    • Single step through some of the first events with the step button: [ht::img images/StepForward24.gif].
    • Start the replay by clicking the play button: [ht::img images/Play24.gif].
    • You should see the events that you just did replay in the same speed at which you first did them.
    • Change the playback speed to [ht::qt 2X] by selecting the button in: [ht::cimg images/CaptureGuiSpeed.gif]
    • Select the [ht::button Skip long delays] checkbox. This allows you to skip over any intervals between events that are greater than one second.
    • If it hasn't run through already press the stop button: [ht::img images/Stop24.gif].
    • Rewind back to the beginning with the restart button: [ht::img images/Rewind24.gif].
    • Remove the data and displays. [ht::refresh]
    • Change the playback speed to [ht::qt 10X] and press the [ht::button Play] button.
    • When the playback has stopped clear the data and displays. [ht::refresh]
