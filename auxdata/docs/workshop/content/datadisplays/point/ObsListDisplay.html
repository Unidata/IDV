The Observation List Display Control shows a textual listing of observation data. We are going to look at some earthquake point data.

  1. [ht::clearDisplaysAndData]
  2. Load some point data.
    • [ht::openDataSourceChooser]
    • In the [ht::dialog Data Source Chooser] click on the [ht::command Files] tab.
    • Go to the point data directory: [ht::filename <%workshop.datadir%>/point/].
    • Select [ht::filename earthquake.nc] (Note: not [ht::filename earthquake.csv])
    • Under the [ht::command Data Source Type] pulldown menu, select "Netcdf Point Data files"
    • Press [ht::button Add Source]
  3. Create the display
    • In the [ht::command Fields] panel select [ht::param Point Data].
    • In the [ht::command Displays] select [ht::display Point Data List] and press [ht::button Create Display]
  4. Modify the displayed fields.
    • Press the [ht::button Select Fields] button
    • We see there are 4 fields shown and one field ([ht::param var]) not shown.
    • Select [ht::param var] and press [ht::button Add]
    • Change the order of the fields - select [ht::param var] and then press the up arrow button.
    • Press [ht::button OK].
  5. Modify the listing.
    • Toggle [ht::button Show Raw Data]
    • Sort the columns by pressing the [ht::param var] column header.
    • Figure out what is the maximum depth of the observations.
  6. Export the data as a text CSV file.
    • Use [ht::menu File Save {Export Table}] in the [ht::display Point Data List] display control.
    • Enter [ht::filename earthquake.csv]
    • Read in this file from the File Chooser. Make sure you set the [ht::command Data Source Type] to "I'm Feeling Lucky"
    • Create another [ht::display Point Data List] display with this CSV data.
