Installing on Windows

Installing RAMADDA involves the installation of these three software packages:

  • IDV
  • Apache Tomcat
  • RAMADDA
We will now go through each of steps in detail.

IDV Installation

  1. Install the latest version of the IDV

Tomcat Installation

  1. Download the Windows Service Installer of the Apache Tomcat core module to your Desktop.
  2. Double click on the apache-tomcat-x.y.z icon. (x.y.z stands for the particular version of Apache Tomcat you downloaded)
  3. Allow the Tomcat installer to run on your computer.
  4. Click Next
  5. Accept License Agreement
  6. Accept the default components. Click Next
  7. Accept default configuration options. Click Next
  8. Choose the Java Virtual Machine in the IDV directory (e.g. C:\\Program Files\\IDV_3.0\\jre)
  9. Accept default installation location. Click Install
  10. Unclick "Run Apache Tomcat" and "Show Readme", and Finish.
  11. Open Windows Explorer and go to C:\\Program Files\\Apache Software Foundation
  12. Right click on Tomcat 7.0 folder and select "Properties", "Security" tab, select "Users", click "Edit"
  13. Select "Users", check "Full control", click OK, click OK again

RAMADDA Installation

  1. Download the RAMADDA WAR (web application archive) file.
  2. Save the RAMADDA WAR file to C:\\Program Files\\Apache Software Foundation\\Tomcat7.0\\webapps
  3. Start your Windows Task Manager(CTRL+ALT+DELETE, and then click Task Manager).
  4. Select the "Services" tab, and click on the "Services" button.
  5. Select the "Apache Tomcat 7", right click and select "Properties".
  6. Go to "Logon" tab, "Local System account", do NOT check "Allow service to interact with desktop".
  7. Go to the "General" tab and click "Start" the service.
  8. In your browser, navigate to http://localhost:8080/repository If you see the following screen, you have successfully installed RAMADDA.

You are now ready to to set up the admin account in RAMADDA.
