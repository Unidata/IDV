METAR and Synoptic data can be accessed in the IDV from ADDE servers. The data available is hourly, each hourly file having all station observations at that time.

  1. [ht::clearDisplaysAndData]
  2. In the [ht::dialog Data Source Chooser] go to the [ht::command Point] tab.
  3. In the [ht::command Server:] entry box, select the remote ADDE server "adde.ucar.edu".
  4. In the [ht::command Data Type:] entry box, select the choice "Surface (METAR) Data" and click on [ht::button Connect].
  5. The IDV will connect to the ADDE server at adde.ucar.edu and download metadata about the available METAR data.
  6. Select [ht::command Absolute times] to see a list of data times that you can get from this server.
  7. The bottom time in the list, the most recent time, should be highlighted. If it is not, click on it.
  8. Hold down the control key and click on the second to last time so that two times are selected.
  9. You could also select [ht::command Use relative times] and select [ht::command Most recent 2 times].
  10. Make sure the [ht::command Layout Model:] is "Observations>METAR".
  11. Click the [ht::button Add Source] button.
  12. Because the [ht::button Create Display] checkbox is selected the IDV automatically creates a point data plot.
