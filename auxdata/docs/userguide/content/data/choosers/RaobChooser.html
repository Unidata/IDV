You can access RAOB data either from remote ADDE servers (using the [ug::chooser Soundings: Remote] chooser, pictured below) or from local files (using the [ug::chooser Soundings: Local] chooser). The only difference between these two choosers is specifying the source of data. You either select an ADDE server and press [ug::button Connect] or you select a file containing RAOB data.

As a local file, the IDV has the capability to handle GEMPAK upper air format data. The GEMPAK upper air files must have the .gem extension to be read by the IDV.

[ht::cimg images/RaobServerChooser.gif {RAOB Chooser}] To select RAOB data you:
    [ug::step {Choose the server or the local file. The list of times will be displayed. In this chooser you select particular stations at particular times.}] [ug::step {Select a set of dates and times that you want to use. To select more than one time use Control-Click or Shift-Click.}] [ug::step {Select one or more stations for the given time using the map. See below for more information on how to use the station map.}] [ug::step {Once selected the station/time combination will be shown in the Selected Stations.}] [ug::step {When done, load the selected RAOB data with the [ug::button Add Source] button.}]
