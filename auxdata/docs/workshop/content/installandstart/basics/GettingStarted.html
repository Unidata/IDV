In this section, we will examine some of the Graphical User Interface (GUI) features of the IDV.

The sections below describe the user interface of the main window of the IDV reference application. First, we'll load in a time sequence of data so we can use some of the menus and widgets in the IDV's GUI and see their effect.

  • From the [ht::menu "Favorites Toolbar"], select the [ht::menu "<%starting.bundle%>"] item from the [ht::menu "Workshop"] menu.
  • When the dialog pops up asking you how you want to load the bundle, click the [ht::command OK] button.
We'll explore the Favorite Bundle mechanism in a later exercise.

View Window

The View Window is where the visualizations of the loaded data are displayed. You can have several of these windows as we'll see in a later exercise. [ht::cimg images/MainWindow.gif]

Dashboard

The Dashboard window holds many of the non "View" windows. It provides a holder for the data choosers, field selector and the display controls as well as quick access to some handy IDV features. [ht::cimg images/Dashboard.gif] Notice that the Dashboard and View windows have identical menu bars. This allows you to control the IDV from either window.
