The IDV can display 2D and 3D grid trajectory as a derived field based on the horizontal and vertical wind components.

  1. [ht::clearDisplaysAndData]
  2. Load some gridded data.
    • [ht::openDataSourceChooser]
    • In the [ht::dialog Data Chooser] tree click on the [ht::command Files] node.
    • Go to the grid data directory: [ht::filename <%workshop.datadir%>/grids/].
    • Select [ht::filename sample_eta_grid.nc]
    • Press [ht::button Add Source]
  3. Create the trajectory control
    • In the [ht::command Fields] panel open the [ht::param 3D grid] node and [ht::param Derived] node, scroll down the [ht::param Derived] node and select [ht::param Grid 3D Trajectory].
    • In the [ht::command Displays] select [ht::display Trajectory Colored By Parameter] and press [ht::button Create Display]
    • After you click the [ht::button Create Display] button, a dialog will pop up to allow you to select the [ht::param Color by parameter]. From the dialog, expand the [ht::param 3D grid] node and select the [ht::param temperature] field and click the [ht::button OK] button.
  4. Create the trajectory display
    • From the control window for this display, change the level to 700 hectopascals using Levels selector.
    • Click the Rectangle radio button as the trajectory intial area option, click and hold the left mouse button and drag a rectangle in the view window.
    • After selecting the level and specifying the initial area, click on the button Create Trajectory to create the display.
  5. Change the trajectory time display
    • Bring up the display control window for the [ht::display Trajectory] display.
    • In the display control window go to the [ht::command Times] tab. Press Time Mode [ht::button Change] button.
    • For End Time choose [ht::button From Animation Time]
    • For Start Time choose [ht::button Relative to End Time]
    • Change the Offset for the start time to be -6 hours.
  6. Try other levels and trajectory intial area options, and do more with grid trajectory display.
