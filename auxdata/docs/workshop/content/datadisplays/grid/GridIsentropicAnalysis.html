Isentropic analysis is a common and effective way to diagnose horizontal and vertical motion of the atmosphere, the combination of 3D visualization feature of the IDV and the isentropic analysis helps us better visualize the dynamic and stability of weather phenomenons.

  1. [ht::clearDisplaysAndData]
  2. Load some gridded data.
    • [ht::openDataSourceChooser]
    • In the [ht::dialog Data Chooser] tree click on the [ht::command Files] node.
    • Go to the grid data directory: [ht::filename <%workshop.datadir%>/grids/].
    • Select [ht::filename sample_eta_grid.nc]
    • Press [ht::button Add Source]
  3. Create a isentropic topography display
    • In the [ht::command Fields] panel open the [ht::param 3D grid] node and [ht::param Derived] node, scroll down the [ht::param Derived] node and select [ht::param Potential Temperature IsoSurface].
    • In the [ht::command Displays] open [ht::display Constant Theta Surface Height Plan View] node, and select [ht::display Isentropic Topography] and press [ht::button Create Display]
    • A dialog will pop up to allow you to set the isosurface value. The default isosurface value is 310 (default). Click the [ht::button OK] button to create the display.
    • After creating the display you can adjust the data range for the selecting color table.
  4. Create a plan view analysis on isentropic surface
    • In the [ht::command Fields] panel open the [ht::param 3D grid] node and [ht::param Derived] node, scroll down the [ht::param Derived] node and select [ht::param Potential Temperature IsoSurface Scalar Analysis].
    • In the [ht::command Displays] open [ht::display 3D Theta Surface] node, and select [ht::display Contours Over Constant Theta Surface] and press [ht::button Create Display]
    • Set the input of the isosurface value to 310.5.
    • After you click the [ht::button Ok] button, a dialog will pop up to allow you to select the [ht::param Parameter] field. From the dialog, expand the [ht::param 3D grid] node and select the [ht::param Temperature@Isobaric surface] field and click the [ht::button OK] button.
    • After creating the display you can adjust the contour interval and apply the smoothing to the resulting display.
  5. Create a vector plan view analysis on isentropic surface
    • In the [ht::command Fields] panel open the [ht::param 3D grid] node and [ht::param Derived] node, scroll down the [ht::param Derived] node and select [ht::param Potential Temperature IsoSurface Vector Analysis].
    • In the [ht::command Displays] open [ht::display Constant Theta Surface Flow Displays] node, and select [ht::display Vectors Colored by Speed over Theta Surface] and press [ht::button Create Display]
    • Set the input of the isosurface value to 310.5.
    • After you click the [ht::button Ok] button, a dialog will pop up to allow you to select the [ht::param voctor] field. From the dialog, expand the [ht::param 3D grid] node and the [ht::param Derived] node and select the [ht::param True Wind vectors] field and click the [ht::button OK] button.
    • After creating the display you can set the skip value to adjust the density of the resulting display.
