An in-depth look at an ISL that is used to generate Google Earth KMZ products. This uses the [gen::islTag import], [gen::islTag procedure], [gen::islTag call] and [gen::islTag output] tags.
  1. To start make an [ht::filename <%workshop.homedir%>/example] directory.
  2. Right click on the link below and save the file to the [ht::filename example] directory.
    example.isl
  3. You will need to create 2 bundles in the [ht::filename example] directory:
    [ht::filename example1.jnlp]
    [ht::filename example2.jnlp]
  4. To run this: 
    idv <%workshop.homedir%>/example/example.isl

Here is the list of bundles: [gen::isl example.isl]

Here is the file that actually does the work. By default this is in the IDV library. [gen::isl generate.isl]
