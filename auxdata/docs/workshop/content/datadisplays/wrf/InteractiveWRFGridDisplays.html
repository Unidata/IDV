Displays for WRF post-processed data.
The following are examples of the different kids of displays available in the IDV. For more information how the display was created, simply click the image; the pages that pop-up use different data sources, but the instructions for creating the displays are the same. Note: Many of the example images used below take advatage of the NASA blue marble background image. If you wish to use blue marble, simply click Display -> Maps and Brounds -> Add Background Image and select Blue Marble.
