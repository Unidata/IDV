Vector components (e.g. winds, ocean currents, mantle plate velocities) can be displayed in the IDV in a variety of ways.

  1. First, let's create a display of wind barbs with geopotential heights.
    • Clear out any existing displays.
    • In the [ht::dialog Field Selector]'s [ht::command Data Sources] panel, select the [ht::dataset NCEP Model Data ETA] data source.
    • In the [ht::command Fields] panel, expand the [ht::param 3D grid] tab. Select the [ht::param geopotential height], the [ht::display Contour Plan View] display and set the [ht::command Level] to be 500 hectopascals, then click the [ht::command Create Display] button.
    • In the [ht::command Fields] panel, expand the [ht::param 3D grid] tab. Scroll down to the [ht::param Derived] tab and expand that. Select the [ht::param {Flow Vectors (from GridRelative_u & GridRelative_v)}] field.
    • In the [ht::command Displays] panel, select [ht::display Wind Barb Plan View], in the [ht::command Level] tab, select 500 hectopascals, then click the [ht::command Create Display] button.
    • The display is pretty cluttered. You can change the skip interval to declutter the display. Try various intervals (found on the 'Display' tab of the dashboard).
    • Toggle between vectors and streamlines and play around with the streamline density slider.
    • Link the two displays by selecting the [ht::menu Edit Sharing {Sharing On}] menu.
    • Change the level and see how the pattern changes.
    • Turn on the time animation.
  2. Now, let's add a cross section display of wind vectors
    • Stop the time animation.
    • In the [ht::dialog Field Selector]'s [ht::command Fields] panel, select the [ht::param {Flow Vectors (from GridRelative_u & GridRelative_v)}] field.
    • In the [ht::command Displays] panel of the [ht::dialog Field Selector], select [ht::display Vector Cross Section] and then click the [ht::button Create Display] button.
    • Move the cross section around and see how the flow changes.
    • Start the animation.
