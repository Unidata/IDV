Linux/Solaris instructions

Note: If you downloaded a file other than idv_<%idv.version%>_linux-i386_installer.sh or saved the installer as a different name, replace the filename listed in the steps below with the name of the file you downloaded.

  1. Open a terminal window.
  2. Change to your home directory or the directory to which you downloaded the installer (if you are not already there).
  3. Run the shell file by typing:

      sh idv_<%idv.version%>_linux-i386_installer.sh
  4. Follow the prompts of the installer using the default selections.
